RU-Wld

The Radboud University WLD provides an interface to the Limburg dialect dictionary. This dictionary contains dialect words, each of which has a dialect identifier. Entries also contain a more generic Lemma.

Learn more »

Getting started

The RU-Wld has been designed by the Technical Service Group of the Humanities Lab at the Radboud University Nijmegen. Those that have been asked to cooperate in adding content can request access. Others are free to browse and look at its contents.

Request access »

Define entries

Dialect entries can be defined straight away by choosing Edit dictionary from the menu above. This brings up one of the "Dictionary Admin" pages.

The Definitions option from the admin menu opens up a page where each of the components and sub-components of entries can be edited separately. This page can be used by those who are more interested in a bottom-up approach to entering the necessary data.
